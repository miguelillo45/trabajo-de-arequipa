TAREA: RESPONDER 5 PREGUNTAS

1) Que es un protocolo?

Es el código que se establece para que el computador solicitante y el que contiene la información solicitada puedan “hablar” un mismo idioma a la hora de transmitir información por la red.

2)¿Que es un querry string HTTP?

Es un término que se utiliza para hacer referencia a una interacción con una base de datos.

3)¿Que es un verbo HTTP?

Es un conjunto de métodos de petición para indicar la acción que se desea realizar para un recurso determinado.

4)¿Qué es un DNS?

DNS es una base de datos distribuida cuya función es traducir los nombres de dominio en las redes.

5)¿Qué es un CDN?

Es un grupo de servidores repartidos en distintas zonas geográficas que aceleran la entrega del contenido web al acercarlo a los usuarios.
